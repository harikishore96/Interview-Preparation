# Arrays
### Initialize,
```
int[] numbers = {10, 20, 30, 40};
int[] numbers = new int[]{10, 20, 30, 40};
int[] numbers = new int[10];
```

### Access an element,
```
int[] numbers = {10, 20, 30, 40};
System.out.println(numbers[0]);
```

### Update an element,
```
numbers[0] = 11;
```

### Length,
```
System.out.println(numbers.length);
```
